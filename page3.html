Sauvegarder projet

Ajoute un fichier spécifique pour préparer la sauvegarde
git add index.html
Ajoute tous les fichiers pour préparer la sauvegarde
git add .
Affiche les modifications qui vont être sauvegardées
git status
Créer une sauvegarde avec un commentaire sur la version
git commit -m "création du fichier index.html"
